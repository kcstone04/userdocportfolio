Why User Documentation?

I believe that knowing how to write effective user documentation is a useful tool for anybody, as it is primarily about efficiently giving easy-to-understand instructions. For the purposes of my major (professional and technical writing in particular), it is crucial for many potential post-graduation jobs, such as writing instruction manuals, or website content writing. I personally hope to get some sort of content writing job, so this class is imperative for that.

Portfolio Overview

This website is a digital portfolio containing all of my major projects from the class ENGL 3814: Creating User Documentation. It is meant to showcase my understanding of the class content, and my ability to create effective user documents. It is also meant to represent all my accomplishments in this class, and serve as an effective capstone to the course as a whole.
